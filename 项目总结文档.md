# 个人博客独立站 - 项目总结文档

## 一、项目概述

本项目是一个完整的个人博客独立站，采用纯静态网站技术（HTML/CSS/JS）开发，部署在 Vercel 平台上，使用自定义域名 `fuqiupu.vercel.app`。项目包含前台展示和后台管理两大模块，支持 Markdown 文章发布、云端数据同步、分类管理等功能。

- **线上地址**：https://fuqiupu.vercel.app/
- **管理后台**：https://fuqiupu.vercel.app/admin.html
- **GitHub 仓库**：https://github.com/Fuqiupu/blog
- **后台登录密码**：`admin123`

## 七、部署与更新指南

### 7.1 首次部署（已完成）

1. 代码推送到 GitHub 仓库 `Fuqiupu/blog`
2. Vercel 连接 GitHub 仓库，自动部署
3. 访问 `fuqiupu.vercel.app` 即可使用

### 7.2 日常更新文章

1. 访问 `https://fuqiupu.vercel.app/admin.html`
2. 输入密码 `admin123` 登录
3. 在"发布新文章"区域填写内容或上传 Markdown 文件
4. 选择分类，点击"发布文章"
5. 文章自动同步到云端，所有设备可见

### 7.3 配置云端同步（首次使用需操作）

1. 登录管理后台
2. 在"云端同步设置"区域输入 GitHub Token
   - 获取方式：GitHub → Settings → Developer settings → Personal access tokens → 生成新 token（勾选 repo 权限）
3. 点击"保存 Token"
4. 点击"同步本地文章到云端"按钮，迁移已有文章

### 7.4 修改管理密码

编辑 `admin.html` 文件，找到以下代码修改密码：

```javascript
const ADMIN_PASSWORD = 'admin123';  // 修改为你想要的密码
```

### 7.5 更新网站代码

```bash
# 克隆仓库
git clone https://github.com/Fuqiupu/blog.git

# 修改代码后提交
git add .
git commit -m "更新说明"
git push origin main

# Vercel 会自动重新部署
```

## 八、关键代码说明

### 8.1 文章数据加载（js/main.js）

首页和文章详情页通过 fetch 异步加载云端文章数据，同时合并本地存储和默认文章：

```javascript
// 从云端加载文章数据
async function loadCloudPosts() {
  const response = await fetch('posts.json?t=' + Date.now());
  if (response.ok) {
    cloudPosts = await response.json();
  }
}

// 获取所有文章（合并云端、本地、默认）
function getAllPosts() {
  const savedPosts = JSON.parse(localStorage.getItem('blogPosts') || '[]');
  const allPosts = [...cloudPosts, ...savedPosts, ...defaultPosts];
  return allPosts.filter((post, index, self) =>
    index === self.findIndex(p => p.id === post.id)
  );
}
```

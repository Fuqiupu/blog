# 个人博客独立站前端设计方案（简易版）

## 一、需求分析

### 1.1 项目概述
创建一个个人博客独立站，采用**纯 HTML/CSS/JS** 实现，无需构建工具，单文件即可运行，支持响应式设计。

### 1.2 技术选型变更
- **原方案**: React + Vite + TailwindCSS（需要 npm 构建）
- **新方案**: 纯 HTML/CSS/JS（无需构建，浏览器直接打开即可运行）

### 1.3 功能需求（精简版）
| 模块 | 功能描述 | 优先级 |
|------|----------|--------|
| 首页 | 博客文章列表、分类导航 | 高 |
| 文章详情 | 展示单篇文章内容 | 高 |
| 关于页 | 作者介绍信息 | 中 |
| 导航栏 | 顶部导航，移动端汉堡菜单 | 高 |
| 页脚 | 版权信息、社交链接 | 中 |

---

## 二、技术架构

### 2.1 项目结构（极简）
```
/workspace/
├── index.html          # 主页面（包含首页 + 文章列表）
├── post.html           # 文章详情页
├── about.html          # 关于页
├── css/
│   └── style.css       # 全局样式（响应式）
└── js/
    └── main.js         # 交互逻辑（导航、数据渲染）
```

### 2.2 技术特点
- **零依赖**: 不需要任何 npm 包或构建工具
- **原生实现**: 使用原生 HTML5 + CSS3 + ES6+ JavaScript
- **响应式**: 使用 CSS Grid + Flexbox + 媒体查询
- **数据驱动**: 使用 JS 对象存储文章数据，动态渲染

---

## 三、页面设计

### 3.1 首页（index.html）
- **布局**: 响应式网格布局
- **内容**:
  - 顶部导航栏（Logo + 菜单 + 移动端汉堡按钮）
  - Hero 区域：博客标题、简介
  - 文章列表（卡片式，响应式网格）
  - 页脚

### 3.2 文章详情页（post.html）
- **布局**: 单栏居中布局
- **内容**:
  - 文章标题、发布时间、分类
  - 文章正文
  - 返回按钮
  - 页脚

### 3.3 关于页（about.html）
- **布局**: 单栏居中布局
- **内容**:
  - 作者头像、简介
  - 技能列表
  - 联系方式
  - 页脚

---

## 四、开发步骤

### 阶段一：清理旧文件
1. 删除 React 相关文件（package.json、vite.config.js、jsconfig.json）

### 阶段二：创建基础结构
1. 创建 `index.html`（首页）
2. 创建 `css/style.css`（全局样式）
3. 创建 `js/main.js`（数据 + 交互逻辑）

### 阶段三：创建其他页面
1. 创建 `post.html`（文章详情页）
2. 创建 `about.html`（关于页）

### 阶段四：样式优化
1. 响应式适配（移动端、平板、桌面端）
2. 基础动画效果（hover、过渡）

---

## 五、响应式设计

| 设备 | 断点 | 布局 |
|------|------|------|
| 移动端 | < 768px | 单栏，汉堡菜单 |
| 平板 | 768px - 1024px | 双栏文章列表 |
| 桌面端 | > 1024px | 三栏文章列表 |

---

## 六、交付标准

1. **零依赖**: 无需安装任何依赖，浏览器直接打开
2. **响应式**: 支持移动端、平板、桌面端
3. **代码规范**: 语义化 HTML，模块化 CSS/JS
4. **兼容性**: 支持 Chrome、Firefox、Safari 最新版本
